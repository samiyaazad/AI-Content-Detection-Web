# AI Content Detection Web Tool

## Problem
People struggle to determine whether an image or video is AI-generated, which can lead to misinformation or confusion.

## Solution
A web tool that:
- Allows users to upload images or videos
- Analyzes content to detect whether it is AI-generated
- Provides confidence scores and explanation of the detection

## Status
Concept / Planning

## Impact
Helps prevent misinformation and educates users about AI-generated content, increasing digital literacy.
